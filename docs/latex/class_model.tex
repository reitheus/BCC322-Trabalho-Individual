\doxysubsection{Detailed Description}
Classe responsável por representar um modelo de simulação. 

A classe \doxylink{class_model}{Model} armazena os sistemas e fluxos que compõem um modelo de dinâmica de sistemas, sendo responsável pela execução da simulação.
