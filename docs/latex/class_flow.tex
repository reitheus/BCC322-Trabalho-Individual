\doxysubsection{Detailed Description}
Classe abstrata responsável por representar um fluxo. 

A classe \doxylink{class_flow}{Flow} define a estrutura básica de um fluxo em um modelo de dinâmica de sistemas, conectando um sistema de origem a um sistema de destino.
